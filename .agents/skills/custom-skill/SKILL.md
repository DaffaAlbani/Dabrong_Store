# Custom Skill

Gunakan skill ini untuk menginstruksikan AI mengenai tugas khusus di workspace ini.

## Cara Menggunakan
Tulis instruksi detail Anda di sini. Ketika AI mendeteksi kecocokan dengan `name` atau `description` di atas, instruksi ini akan otomatis dimuat.
